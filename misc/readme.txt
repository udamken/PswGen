How to create the layered ICO file?

Open PswGen.svg in Inkscape
	Export to PswGen_512x512.png

Open PswGen_512x512.png in Gimp
	Script-Fu > Phota Save Icons > PNG

Open PswGen-512.png in Gimp
	File > Open as Layers > PswGen-16.png .. PswGen-256.png
	Save as ... PswGen.xcf
	Export File as ... etc/de/dknapps/pswgen/PswGen.ico
